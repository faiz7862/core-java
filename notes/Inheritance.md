Inheritance

*

    The class from where u r deriving is known as super class (Parent class).

    The class to which we are deriving is known as sub class (child class).

    We can derive properties from one class to another class using the keyword "extends".

    Deriving properties from one class to another class is known as inheritance.

Types of inheritance

* Single inheritance

* Multilevel inheritance



* Multiple inheritance

* Hierarchial inheritance

* Hybrid inheritance
